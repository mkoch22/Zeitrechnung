# Zeitrechnung
Dieses Programm kann zu einer Uhrzeit eine beliebige Anzahl von Minuten addieren. 

## Features:

- man kann selber entscheiden was man berechnen will (beliebige Zahlen)
- einfaches Programm
- kann auch 4-stellige Zahlen berechnen
